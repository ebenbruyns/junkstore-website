<h2 class="centered-header">How to Install the GOG Extension</h2>

<p style="text-align: center;">Add GOG library integration to your Junk Store Decky plugin for DRM-free gaming access.</p>

<div class="install-guide">
<ol>

<li>
<strong>Install Junk-Store</strong>
<p>Make sure you have the latest version of Junk-Store installed from the Decky Store.</p>
</li>

<li>
<strong>Switch to Desktop Mode</strong>
<p>Download the latest GOG extension ZIP file from the portal, then right-click the ZIP file and choose "Extract Here".</p>
<img src="/assets/images/gogext/extracthere.png" alt="Extract the GOG extension ZIP file" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Launch the Installer</strong>
<p>Double-click the extracted file to launch the installer.</p>
<img src="/assets/images/gogext/launchinstaller.png" alt="Launch the GOG extension installer" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Confirm Installation</strong>
<p>Follow the prompt to install the GOG extension.</p>
<img src="/assets/images/gogext/installerprompt.png" alt="Click yes to run the installer" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Return to Game Mode</strong>
<p>Once installed, switch back to Game Mode.</p>

<div style="background: rgba(255, 193, 7, 0.1); border-left: 4px solid #ffc107; padding: 1rem 1.5rem; margin: 1rem 0; border-radius: 0 6px 6px 0;">
<strong>Note:</strong> You will need to reinstall the dependencies after installation.
</div>

<img src="/assets/images/gogext/installdeps.png" alt="Install dependencies in Junk Store" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Verify the GOG Tab</strong>
<p>The GOG tab should now be visible in Junk-Store. Log into GOG using your GOG account credentials.</p>
<img src="/assets/images/gogext/jstabs.webp" alt="GOG tab visible in Junk Store" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Test with a Simple Game</strong>
<p>Install a basic game like <em>Absolute Drift</em> to confirm the extension is working correctly.</p>
</li>

</ol>
</div>
