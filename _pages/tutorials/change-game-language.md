<h2 class="centered-header">How to Change Game Language Settings</h2>

<p style="text-align: center;">Change your Epic, GOG, or Amazon game language settings directly in Game Mode - no desktop switching required.</p>

---

<h3 style="margin-top: 2rem;">Method 1: Environment Variables (for individual games)</h3>

<div class="install-guide">
<ol>

<li>
<strong>Open the Settings Menu</strong>
<p>From the game page in Junk Store, click the <strong>⚙️ Settings menu</strong>.</p>
<img src="/assets/images/tutorials/change-language/CogGamePage.webp" alt="Settings menu on game page" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Select Game Configs</strong>
<p>Select <strong>Game Configs</strong> to access the configuration screen.</p>
<img src="/assets/images/tutorials/change-language/GameConfi.webp" alt="Game configuration screen with dropdowns" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Change Dropdown to All</strong>
<p>Change the dropdown to <strong>All</strong> to see all available options.</p>
</li>

<li>
<strong>Select Your Language</strong>
<p>Click the <strong>ENVIRONMENT</strong> dropdown, click on <strong>LANG</strong> and select your preferred language.</p>
<img src="/assets/images/tutorials/change-language/LangDropdown.webp" alt="Language dropdown with options" width="600" height="338" loading="lazy" />
<img src="/assets/images/tutorials/change-language/LangSelection.webp" alt="Language selection example" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Save Changes</strong>
<p>Press <kbd>X</kbd> to save your changes.</p>
</li>

</ol>
</div>

---

<h3 style="margin-top: 2rem;">Method 2: Installation Settings (affects game installation language)</h3>

<div class="install-guide">
<ol>

<li>
<strong>Access Installation Settings</strong>
<p>Select the appropriate tab for installation settings, then click the <strong>⚙️ Settings menu</strong> from the game page in Junk Store.</p>
<img src="/assets/images/tutorials/change-language/CogTab.webp" alt="Installation settings tab" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Configure Language Preference</strong>
<p>Access the installation settings and select your preferred language.</p>
<img src="/assets/images/tutorials/change-language/LangDropdown.webp" alt="Language dropdown with options" width="600" height="338" loading="lazy" />
<img src="/assets/images/tutorials/change-language/LangSelection.webp" alt="Language selection for installation" width="600" height="338" loading="lazy" />
</li>

<li>
<strong>Download with Selected Language</strong>
<p>This will download and install the game with your selected language files.</p>

<div style="background: rgba(255, 193, 7, 0.1); border-left: 4px solid #ffc107; padding: 1rem 1.5rem; margin: 1rem 0; border-radius: 0 6px 6px 0;">
<strong>Note:</strong> If changing language after game installation, update the game from the ☰ Slider menu to pull in new language packs.
</div>
</li>

</ol>
</div>

<p style="margin-top: 1.5rem;">Your game will launch in the selected language - all managed from Game Mode with no additional tools needed.</p>
