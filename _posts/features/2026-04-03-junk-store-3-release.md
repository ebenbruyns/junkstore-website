<h2 style="text-align: left !important; margin-left: 0;">What's New in 3.0</h2>

### Built-in File Manager

Need to copy a save file, move a mod, or just poke around your game folders? Now you can do it all from Game Mode. No more switching to Desktop Mode for basic file operations.

- Access game folders directly
- Browse other Junk Store devices on your network
- Quick prefix navigation
- Manage SD cards

### Disk Analyser

Running low on space and not sure what's hogging it all? The new Disk Analyser shows you exactly what's eating your storage with a visual breakdown you can drill into.

- Identify large files at a glance
- Visual storage breakdown
- Easily drill down into directories

### Process Manager

Keep tabs on what's running on your Deck. The new Process Manager lets you see and manage active processes without leaving Game Mode.

### Improved GOG Support

DOS games, ScummVM titles, and native Linux games from GOG now work better than ever. We've put a lot of work into making these run smoothly out of the box.

### SteamGridDB Built-in

Update your game artwork right from Game Mode. Browse and apply custom covers, backgrounds, and logos without ever touching Desktop Mode.

<h2 style="text-align: left !important; margin-left: 0;">Everything Else You Love</h2>

All the features that made Pro great are still here and better than ever:

- **All your stores in one place**: Epic, GOG, Amazon Prime Gaming, and itch
- **Download queue management**: Pause, resume, and reorder across all stores
- **One-click dependencies**: VC++, DirectX, .NET installed without the hassle
- **Language selection**: Pick your preferred language for supported games
- **Selective DLC**: Choose which DLC to install
- **Diagnostics framework**: Automatic problem detection
