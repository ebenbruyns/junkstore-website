{%- comment -%} "In case you missed it" callout(s). Build-time, automatic, no admin-app or per-post changes needed. Opt out on any single post with `prev_callout_disabled: true` in frontmatter. Two modes: * Compat testing posts (Weekly Update / Backlog Testing / Game Retests) get a compact "Catch up on recent testing" block linking the latest post in EACH of the three compat categories, excluding the current post. A category with no other post is silently skipped. * Every other post type keeps the original single previous-in-series link. Note: Weekly posts may also carry a hand-written "In case you missed it" line in the body (e.g. pointing to dev updates) — that's separate content, authored per-post, and is not affected by this include. site.posts is newest-first, so the first matching post (other than the current page) is the most recent one in that category. {%- endcomment -%} {%- assign this_cat = page.categories | first -%} {%- if this_cat and page.prev_callout_disabled != true -%} {%- assign trigger_cats = "Weekly Update|Backlog Testing|Game Retests" | split: "|" -%} {%- assign is_compat = false -%} {%- for c in trigger_cats -%} {%- if this_cat == c -%}{%- assign is_compat = true -%}{%- endif -%} {%- endfor -%} {%- if is_compat -%} {%- comment -%} Gather the latest post in each compat category (not this page). {%- endcomment -%} {%- assign weekly_pick = nil -%} {%- assign backlog_pick = nil -%} {%- assign retest_pick = nil -%} {%- for post in site.posts -%} {%- if post.url != page.url -%} {%- if weekly_pick == nil and post.categories contains "Weekly Update" -%}{%- assign weekly_pick = post -%}{%- endif -%} {%- if backlog_pick == nil and post.categories contains "Backlog Testing" -%}{%- assign backlog_pick = post -%}{%- endif -%} {%- if retest_pick == nil and post.categories contains "Game Retests" -%}{%- assign retest_pick = post -%}{%- endif -%} {%- endif -%} {%- endfor -%} {%- if weekly_pick or backlog_pick or retest_pick -%}
