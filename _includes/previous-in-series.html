{%- comment -%} "In case you missed it" callout linking to the previous post that shares this post's first category (the series). Build-time and automatic: works for every post type (Weekly Update, Dev Update, News, ...) with no per-post or admin-app changes. site.posts is newest-first, so the first same-category post encountered after the current one is the next-older post in the series. {%- endcomment -%} {%- comment -%} Opt out on any single post with `prev_callout_disabled: true` in frontmatter. {%- endcomment -%} {%- assign this_cat = page.categories | first -%} {%- if this_cat and page.prev_callout_disabled != true -%} {%- assign prev_in_cat = nil -%} {%- assign seen_current = false -%} {%- for post in site.posts -%} {%- if post.categories contains this_cat -%} {%- if seen_current and prev_in_cat == nil -%}{%- assign prev_in_cat = post -%}{%- endif -%} {%- if post.url == page.url -%}{%- assign seen_current = true -%}{%- endif -%} {%- endif -%} {%- endfor -%} {%- if prev_in_cat -%} In case you missed it {{ prev_in_cat.title }} {%- endif -%} {%- endif -%}
